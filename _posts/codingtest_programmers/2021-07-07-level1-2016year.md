## 1차 시도

```java
import java.text.SimpleDateFormat;
import java.util.Date;
import java.util.Locale;

class Solution {
    public String solution(int a, int b) {
        String answer = "";
        
        Date initDate = new Date();
        initDate.setYear(116); //2016년
        initDate.setMonth(a-1); //월
        initDate.setDate(b); //일

        SimpleDateFormat sdf = new SimpleDateFormat("E", new Locale("en"));
        answer = sdf.format(initDate).toUpperCase();

        return answer;
    }
}
```

### 생각

- 날짜 처리에 대한 방법만 알면 금방 할 수 있는 코드 인 것 같다.
- Date 가 오래된 것이라 안쓰려했는데 기억이안난다.
- 그냥찍으니 Locale 가 ko 라 아무리해도 '화' 라고 나온다. locale 을 적용했다.


## 첨삭

like 가 많고, 가장 java 스러운 코드

```java
import java.time.*;
class Solution {
  public String solution(int a, int b) {
      return LocalDate.of(2016, a, b).getDayOfWeek().toString().substring(0,3);
  }
}
```

### 검토
- 되게 단순하고 간단하고 심플하고 완벽한 코딩같다

### 생각

- Java 하위 버전에서 Date의 문제점 Month의 -1 하는 부분이 맞지않아 버전업되면서 LocalDate 권장을 하며 -1 하는 부분도 없어진것을 알고있지만, 활용하지 못했다.
- localDate 활용을 위해서 좀 찍어봐야겠다.

```java
System.out.println(LocalDate.of(2016, 5, 24).getDayOfWeek().toString().substring(0,3));
System.out.println(LocalDate.of(2016, 5, 24).getDayOfWeek().toString());
System.out.println(LocalDate.of(2016, 5, 24).toString());

//결과
// TUE
// TUESDAY
// 2016-05-24
```
