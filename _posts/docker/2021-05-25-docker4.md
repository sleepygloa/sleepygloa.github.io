DockerHub 에 우리가 만든 컨테이너 이미지들을 등록 할 수 있는데, 이것들을 비공식과 공식(official)로 사용이 가능하다. 공식 이미지가 된다면, 다른사람과 공유를 할 수 있고, 특히 우리가 자주 이용하는 OS나 Lib 관련은 이미 공식으로 등록이 되어있어 명령어만 이용해서 다운(pull) 받을 수 있다.


- 이미지 다운받기 : docker pull
예시 : Centos
  - 프로그램명:버전
  - -a  프로그램명 (모든 태그이미지 다운)
```
docker image pull centos:7 
docker image pull -a centos
```
- 이미지 검색: docker search [옵션] 검색키워드
```
docker search ubuntu
```

- 이미지 삭제 : docker image rm [옵션] 이미지명 [이미지명] 
- 사용하지 않는 이미지 삭제 : docker image prune [옵션]

- docker hub 로그인 : docker login [옵션] [서버]

- docker hub 에 이미지 업로드 : docker image push 이미지명[:태그명]

- dicker hub 로그아웃 : docker logout [서버명]
