## 2. 주요기능
### 페이지 전환 스크립트
```javascript
window.onload = function(){
    prev_button = document.querySelectorAll("button")[0];
    next_button = document.querySelectorAll("button")[1];
    
    totalNum = album.length;
    // alert(totalNum)

    prev_button.addEventListener("click", function(){
        if(pageNum > 0){
            pageNum --;
        }else{
            pageNum = totalNum -1;
        }
        pageChangeFunc();
    })

    next_button.addEventListener("click", function(){
        if(pageNum < totalNum-1){
            pageNum ++;
        }else{
            pageNum = 0;
        }
        pageChangeFunc();
    })

    //최초실행
    // pageNum = 2;
    pageChangeFunc();
}

//여기서 모든 것을 한다.
function pageChangeFunc(){

    for(var i=0; i<totalNum; i++){
        if(pageNum == i){
            //현재 컨텐츠(페이지)
            album[i].classList.add("active");
        }else{
            album[i].classList.remove("active");
        }
    }
}
```

### 

```html
      <ul class="pointWrap">
        <li></li>
        <li></li>
        <li></li>
        <li></li>
      </ul>
```
ul 태그 안에 li 를 보여줄 수만큼 넣는데, 내용은 굳이 넣을 필요가 없다. CSS로 이미지를 생성함.

```css
.pointWrap {
  position: fixed;
  right: 3vw;
  top: 40%;
  z-index: 100;
}

.pointWrap li {
  list-style: none;
  width: 12px;
  height: 12px;
  margin-top: 15px;
  cursor: pointer;
  background-color: #999;
  border-radius: 50%;
  transition: background .3s ease-out;
}

.pointWrap li.active {
  background-color: #000;
}
```
```.pointWrap``` 이 가장 위에 나올수 있게 z-index : 를 높게 작성하고, 페이지가 전환되어도 현재 이동하지 않게 fixed

```.pointWrap li``` 은 list 태그의 점 을 제거 ```list-style: none;``` 하고 사각 영역으로 표현되면서 radius 를 주어서 원처럼 보이게 한다. transition 을 주어서 변경시 부드럽게 처리되도록 함.

```.pointWrap li.active``` li 가 활성화 될 때, 활성화 표시를 하여 보이게 한다.

## 2. 페이지의 크기 지정
  
```css
body {
overflow-x: hidden;
background-color: #fff;
}

section {
position: relative;
height: 100vh;
width: 100vw;
}
```

```body``` 의 가로 스크롤은 제거하고, 배경은 흰색으로 만든다. 
```section``` 보여줄 페이지는 section 단위로 위아래로 이동하면서 보여줄것이기 때문에 화면 가득 차게 ```100vh``` 로 지정하고, 여러 section 이 순서대로 쌓여서 보여줄 것이기 때문에 ```positoin: relative;``` 로 지정한다.

## 3. 스크롤시 점4 위치 표시 및 선택시 해당 위치로 이동
이번에는 JS 를 추가해야한다.

```javascript
//https://www.nike.com/kr/ko_kr/w/men/fw/lifestyle
window.onload = function(){
  var section = document.getElementsByTagName("section");
  var pointBtn = document.querySelectorAll(".pointWrap li");
  var pageNum = 0;
  var totalNum = section.length;

  for( var i = 0; i < pointBtn.length; i++ ){
    (function(idx) {
        pointBtn[idx].onclick = function() {
            pageNum = idx;
            pageChangeFunc();
            window.scrollTo({
              top: section[pageNum].offsetTop,
              behavior: 'smooth',
            })
        }
    })(i);
  }

}
```

화면에 있는 점4 ```.pointWrap li``` 를 찾고 해당 갯수만큼 하나씩 찾아서 클릭이벤트를 추가해준다.
중간의 ```pageChangeFunc();``` 은 2,3 군데에서 사용하기 때문에 function을 따로 뺏고, 아래에서 작성해준다.
클릭시 현재 페이지 정보를 저장 ```pageNum = idx;``` 하고 스크롤 기능이 ```smooth``` 하게 작동한다.

위에서 뺀 function
```javascript
//페이지 변경
function pageChangeFunc(){
  for(var i=0; i<totalNum; i++){
    section[i].classList.remove("active");
    // section[i].className = "";
    pointBtn[i].classList.remove("active");
  }
  section[pageNum].classList.add("active");
  pointBtn[pageNum].classList.add("active");
  
}
```
표현한 페이지(section)와 점4(pointBtn) 의 활성화를 모두 제거하고, ```section[i].classList.remove("active");```, ```pointBtn[i].classList.remove("active");```
선택된 페이지와 점4를 활성화한다. ```section[pageNum].classList.add("active");```, ```pointBtn[pageNum].classList.add("active");```

```javascript
//페이지 로드되면 바로 실행
pageChangeFunc();
```

페이지가 로딩되었을때, 제일 처음 section 이 선택된 것처럼 보이기 위해 코드 제일 마지막에 ```pageChangeFunc();``` 을 선언한다.

이때 잠시 스크롤 이동시에도 티가 안나기 때문에 section 안에 내용을 추가해주자.
