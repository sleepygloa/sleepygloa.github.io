# 1. TOC:Table of Contents

TOC는 H1~H6의 헤더 목록을 표시해주는 기능이다. minimal-mistakes에서는 TOC를 오른쪽 사이즈 바에 표시해주는 기능을 제공한다.


```
toc: true
```


TOC 사용하려면 해당 포스트의 yml에 위와같이 toc를 설정해준다.

```
toc_sticky: true

```
sticky 성격을 추가하려면 yml에 위와같이 toc_sticky도 같이 설정해준다. sticky 설정은 TOC는 사이드바에 고정하는 역할을 한다. 페이지가 화면에서 넘어가면서 TOC가 같이 넘어가지 않고 고정해주는 역할이다.

TOC의 적용 결과는 예제 페이지 https://devinlife.github.io/blog/blog-plan/에서 확인해보자. 예제 페이지는 commit c48ecb7으로 적용하였으니 이 수정사항을 확인해보자.

toc 설정 하나를 더 확인해보자.



```
toc_label: "페이지 주요 목차"

```
toc_label은 TOC의 제목을 직접 설정할 수 있게 해준다. 기본 문구는 “On this page”이다. 사용하고 싶은 문구가 있으면 이를 yml에 넣으면 된다.
