문의 작성

서비스 제공절차

  1. 자료 전달 & 초기 상담
    고객이 프로젝트 개요, 레퍼런스 자료 등을 전달하고 간단한 상담을 진행합니다.
  2. 견적 및 기간 안내 + 계약 체결
    견적서 및 개발 일정 안내 하고, 계약을 체결한 뒤 프로젝트를 채팅/줌/(필요시 대면) 미팅을 진행합니다.
  3. 기획/디자인 협의(AI)
    전달 받은 기획서, UI/UX 시안, 기능명세서 를 검토하고, 내용을 확정합니다.
    * 시안을 통한 AI 기반 디자인이 생성되므로 인력비용이이 크게 절감됩니다.
  4. 초기 개발(시제품) & 1차 피드백
    간단한 프로토타입을 구현하여 공유하고, 기능·디자인 1차 피드백을 받습니다.
    * 프로토타입을 빠르게 노출해 고객 요청사항이 반영되고 있는지 확인합니다.
  5. 상세 기능 개발 + 테스트/검수
    전체 기능을 구현하고 QA(버그 수정, 성능 최적화)를 거쳐 2차 피드백을 통해 보완합니다.
  6. 배포/서비스 안정화
    운영 서버에 배포하고 일정 기간 모니터링을 진행해 서비스가 안정화될 수 있도록 지원합니다.
  7. 인수인계
    소스 코드·관리 계정 전달, 운영 매뉴얼 안내 등 최종 인수인계를 진행합니다.

우리의 특별 혜택

  • 하자보수 평생 무료!
    계약 종료 이후에도 오류(버그)가 발견되면, 동일 환경/코드 기준으로 무상 보수를 제공합니다. (※ 단, 서버 변경·제3자 코드 수정 등은 제외)
  • 평생 유지보수 무료!
    간단한 수정(텍스트·이미지 교체, 레이아웃 소폭 수정)은 무료, 새로운 기능 추가는 유료로 진행합니다.
  • 문제없는 라이센스
    무료 라이센스·이미지·오픈소스만 사용하므로, 상업적 이용에 대해 추가 저작권 문제가 없습니다.
  • 언제든지 쓴소리 환영
    강렬한 피드백을 받을 준비가 되어있습니다. 커피라도 사드려요!
  • 도메인·호스팅 비용 별도
    도메인·호스팅은 고객 사정에 맞춰 별도 비용이 청구될 수 있습니다.
  • 24시간 내 피드백 약속
    유지보수·하자보수 요청 시, 1영업일 내 1차 답변을 드립니다.

    - 주니어/미드 레벨 투입 가능

  • 계약 방식
    - 계약 기간 단위 결제, 프리랜서 또는 세금계산서 증빙
    - 구체 범위·공수는 계약 전 협의

유지보수/운영 안내

  • 소규모/중급 프로젝트

    - 신규 기능 추가는 건별 협의
  • 운영 지원
    - 서버 모니터링, 보안 패치, 정기 업데이트 등
    - 긴급 오류 시 즉시 대응(주말/야간은 별도 협의)
  • 대규모 플랫폼(인력 투입 대상)
    - 운영/유지보수를 위한 개발자·DevOps 투입
    - 서버 확장, 모니터링 등 별도 계약

개발 범위

  • 기본적으로 제공되는 기능
    대상: 기업/개인 홈페이지, 간단한 페이지
    제공 가능 기능:
    • 간단 기획 + AI 기반 디자인 + 홈페이지 제작
    • 회원가입/로그인/게시판 구현(기본 신규/수정/삭제/조회)
    • 반응형 웹 (PC/모바일 최적화)
  • 추가적으로 제공되는 기능
    예: 권한관리, 일정관리, 대댓글 게시판, 물류서비스, 배송서비스 등등
    제공 가능 기능:
    • 기본기능에 대한 내용은 모두 적용됩니다.
    • 상세 기획 + AI 기반 디자인 + 웹/앱 연동 + 설계서 상세 작성
    • 외부 API 연동 (지도, SNS간편회원가입/로그인 등등)
    • 클라우드 인프라 연동(AWS, Azure, GCP 등)
  • 고급/대규모 플랫폼
    대상: 포털, 전문 SaaS(Subscription), 대규모 데이터 처리(빅데이터/AI), 멀티테넌트, 초고트래픽 e-commerce 등
    개발 서비스 제공: 없음 (인력 투입만 가능)
    • 고객 측에서 프로젝트 전반을 리딩
    • 당사는 개발/디자인 인력 파견, 월 단위 계약

참고사항

  • 공수·비용 산정
    - 소규모/중급 프로젝트: 기능 요구사항에 따라 **수백만 원**대 협의
    - 대규모 프로젝트: 직접 개발은 하지 않으며, 인력 투입 시 월 단위 협약
    - 자세한 금액은 화면 수, 기능 복잡도, 스택, 기간 등에 따라 건별 산정
  • 하자보수 기준
    - 최초 세팅된 환경(호스팅/서버) 및 동일 코드 기준으로 무상
    - 환경 변경 시 건별 비용
  • 간단한 수정 범위
    - 텍스트/이미지 교체, 한 페이지 이하의 미세 수정
    - 범위를 넘어서는 경우 건당 추가 견적
  • 도메인/호스팅 비용
    - 고객이 원하는 업체(예: AWS, 카페24 등) 선택 가능
    - 월 요금 별도
  • 긴급 대응·주말/야간
    - 긴급 오류(서버 다운 등) 발생 시 즉시 조치, 주말/야간은 별도 요율
    - 사전 SLA 계약 시 안정적인 대응 가능
